TMS Codex Skill

An installable Codex skill that turns a blank teacher folder into a local Teaching Management System workspace, then supports the daily plan-present-create-form-read-evidence-log loop.

Install

Install the TMS setup skill from https://github.com/diddypopdiddy/tms-codex-skill/tree/main/skills/tms-setup

After installation, restart Codex.

First Run

Use $tms-setup to get started.

Run this inside an empty teacher workspace folder. Codex will create the TMS structure and begin the setup interview.

Daily Use

Use $tms-setup to plan tomorrow.
Use $tms-setup to make an exit ticket.
Use $tms-setup to analyze exit ticket responses.
Use $tms-setup to log what happened today.

The same skill continues after setup, using the local project files as the teacher's source of truth.

Source-backed

Uses uploaded curriculum, schedule, policy, and course files before making school-specific claims.

Student-facing

Creates live Google Forms and linked response Sheets when authenticated Google access is available.

Policy-aware

Keeps raw student evidence in durable approved systems and writes back aggregate summaries by default.

This is not an SIS, LMS, gradebook, or official student-record system. It is a local planning and teaching-management workspace for Codex.
