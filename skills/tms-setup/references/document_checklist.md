# Teacher Document Checklist

Ask for documents in practical batches. Do not make upload completion a blocker for first planning.

## Minimum Useful Setup

- Course name or subject.
- One class period or section.
- Approximate period length.
- Current unit/topic.
- Teacher's preferred lesson rhythm.

## High-Value Uploads

- Syllabus or course outline.
- Curriculum map or pacing guide.
- Unit plan.
- Rubrics.
- Current assignment list.
- Bell schedule.
- School calendar.
- LMS export or assignment list.
- Equipment/lab rules.
- Media-release or publication rules.
- District AI/data-use policy.

## Optional Later Uploads

- Prior lesson slides.
- Existing handouts.
- Vocabulary lists.
- Anchor charts.
- Sample student work with identifiers removed.
- Assessment data in de-identified or approved form.

## Upload Guidance For Teachers

Use plain language:

```text
Upload the syllabus or course outline first. If you do not have that handy, paste the course name, current unit, and what students did last class.
```
